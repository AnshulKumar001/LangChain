## 🔹 What is LangChain?

LangChain is a framework used to build applications powered by **Large Language Models (LLMs)**.

It helps connect AI models with data, tools, and workflows to create intelligent applications.

---

## 🔹 Why LangChain?

⚡ Simplifies LLM application development  
🔗 Connects AI with tools & data  
🧠 Enables intelligent automation  
🤖 Useful for building AI assistants
